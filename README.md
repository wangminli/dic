dic
====

linux命令行下翻译工具

---
#### 安装

```
wget https://raw.githubusercontent.com/wangminli/dic/master/dic.py
sudo mv ./dic.py /usr/bin/dic
sudo chmod +x /usr/bin/dic
```

#### 使用

```
dic hello
################################### 
#  hello 你好 (U: hɛˈlo, hə- E: hə'ləʊ; he- )
#  n. 表示问候， 惊奇或唤起注意时的用语
#  int. 喂；哈罗
#  n. (Hello)人名；(法)埃洛
################################### 
```
